# "Разработка веб-приложения для повышения эффективности и автоматизации продаж электротоваров"
- Проект выполнен в рамках выпускной квалификационной работы по специальности — "Программная инженерия" (09.03.04).
- Университет КубГТУ. Факультет информационных технологий и кибербезопасности
  
## Технологии проекта:
  > ### Front-end: Next.js, TypeScript, React.
  > ### Back-end: PHP 8.3 + composer.
  > ### БД: MySQL.
  > ### Сервер: Apache/Nginx + Node.js через прокси.
